## Google Chrome Profiles

- GoogleAImode [Chrome Profiles sorting](https://share.google/aimode/LKci2DuULsZ81knBD)
- chrome://settings/manageProfile
- Set Name your Chrome profile (secondary sort but first displayed)
- https://myaccount.google.com/personal-info?utm_source=chrome-profile-chooser
- Set FirstName LastName (primary sort put in "(name)")

## Set Sorting process
- Set the manageProfile name to a short domainname-user format
- Example: ghadmin@horseoff.com
  - chromeProfileName: ho-ghadmin
  - Set FirstName: ho-ghadmin
  - Set LastName: horseoff
  - Set DisplayName: ho-ghadmin horseoff
- Verify Sync is on
- Document / Verify User in  [cat2net](https://docs.google.com/spreadsheets/d/1LdyZlFieSd_1APTbG0QahfwZgqBaA9PigO9_5SPSkmk/edit?gid=1959043129#gid=1959043129)
